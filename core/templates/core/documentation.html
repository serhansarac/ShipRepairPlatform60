📖 Platform Documentation

Introduction

Welcome to the Ship Repair Platform. This documentation provides information on how to use the platform efficiently.

🚀 Features

  • 🔹 Ship Operators can compare and predict repair services.
  • 🔹 Repair Companies can manage and offer their services.
  • 🔹 Admins have access to analytics, payments, and service management.

🛠 How to Use?

  1. 📝 Register as a Ship Operator or Repair Company.
  2. 🔍 Navigate through services and select what you need.
  3. 📊 Use the prediction tool for cost estimation.
  4. 💳 Admins can manage invoices and analytics.

📞 Contact

If you have any questions, please contact support@shiprepairplatform.com
